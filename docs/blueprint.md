# **App Name**: NailDesign Pro

## Core Features:

- Dynamic Landing Page Content: Fetch and display content from Firestore including hero sections, benefits, modules, testimonials, FAQs, and footer information to dynamically populate the landing page.
- Image and PDF Storage: Store and retrieve images (instructor, logo, modules, testimonials) and PDFs (support, materials, EPIs, drills) from Firebase Storage.
- Real-Time Availability: Tracks purchase notifications and updates `percentual_vagas` to represent remaining availability.
- Checkout Redirect: Integrate a call-to-action button that redirects users to a specified checkout URL for purchase.

## Style Guidelines:

- Primary color: Deep rose (#C72A7B) for a fashionable, exciting feel.
- Background color: Pale rose (#F8E8EF) to complement the primary color without overpowering the UI.
- Accent color: Soft violet (#994EAC) to create visual interest.
- Headline font: 'Belleza' sans-serif, aligned to fashion, art, and design, suitable for headlines.
- Body font: 'Alegreya' serif for an elegant and readable feel in longer texts.
- Use clean, modern icons that represent nail design concepts and course modules.
- Employ a clean and modern layout with clear content hierarchy to guide users through the landing page effectively.
- Implement subtle animations and transitions to enhance user engagement and provide visual feedback.
